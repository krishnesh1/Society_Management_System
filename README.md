# Society Management System

A full-stack web application built to manage residential societies and apartment complexes. It provides a centralized dashboard to track daily activities, bills, community discussions, and visitors. 

## Features
* **User Management:** Handle resident profiles and admin access.
* **Announcements & Events:** Broadcast important notices and upcoming community events.
* **Billing System:** Track maintenance bills and payment statuses.
* **Complaint Resolution:** Residents can raise issues (plumbing, electrical, etc.), and admins can update their status.
* **Visitor Tracking:** Keep a log of expected and past visitors.
* **Community Polls:** Create polls for community decision-making.

## Tech Stack
* **Frontend:** React.js (Vite), Tailwind CSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Authentication:** JSON Web Tokens (JWT)
